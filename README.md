# Master Thesis - Stable Diffusion

This repository contains the code, data, and documentation for my Master's thesis on Stable Diffusion for BayesFlow. Below is a brief overview of the repository structure and its contents.

## Repository Structure

- **data/**: Contains datasets used in the experiments.
  - `MNIST/`: Raw MNIST dataset files.
  - `oxford-iiit-pet/`: Oxford-IIIT Pet dataset with annotations and images.
- **docs/**: Documentation and notes related to the thesis.
- **experiments/**: Experiment configurations, checkpoints, and results.
- **scripts/**: Scripts for training and sampling models.
- **src/**: Source code for models, data processing, and utilities.

## How to Use

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run experiments using the scripts in the `scripts/` directory.
3. Refer to the `docs/` folder for detailed notes and explanations.
